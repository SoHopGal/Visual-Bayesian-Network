# Visual-Bayesian-Network
A small project of Visual Bayesian Network in course Artificial Intelligence Algorithms


**Bayesian Network Interactive Calculator**

**Artificial Intelligence – Bonus Assignment**

Student: Gal Argov Sofer

This application demonstrates Bayesian Network inference using the Enumeration Algorithm.

**Features:**
*   Bayesian Network visualization
*   Interactive evidence selection
*   Query variable selection
*   Exact Enumeration Algorithm
*   Intermediate calculations
*   Joint probability computation
*   Normalization
*   Complexity analysis

**How to Run**
1.   Run all notebook cells.
2.   The Bayesian Network will be displayed.
3.   Select:
*   Query Variable
*   Evidence
4.   Click Calculate.
5.   The application displays:
*   Posterior probability
*   Enumeration steps
*   Joint probabilities
*   Normalization
*   Complexity analysis
